Flask Message Board

這是一個使用 Flask + SQLite 製作的簡易留言板系統。
使用 Flask 建立 Web 應用，實作完整 CRUD 流程（Create / Read / Update / Delete）

功能
- 新增留言（Create）
- 查看留言（Read）
- 修改留言（Update）
- 刪除留言（Delete）

技術
- Python
- Flask
- SQLite
- HTML (Jinja2)

專案說明
使用者透過表單送出資料（POST），
後端接收後寫入資料庫，
再透過 redirect 回列表頁顯示結果。

此專案主要用來練習基本 CRUD 與 Web 資料流程。
